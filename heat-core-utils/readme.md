CORE module of HEAT (Hotels.com Engine Architecture for Testing)
====

**Features:**

- HTTP, HTTPS and POST, GET support
- JSON based parameter handling
- Logging with SLF4J

**Used technology:**

- [RestAssured](https://code.google.com/p/rest-assured/)
- Java
- TestNG
- Mockito
